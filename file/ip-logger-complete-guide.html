What's This All About?

An IP logger (some people call it an IP tracker or IP grabber) is basically a tool that lets you see someone's IP address when they click a link you send them.

This page has all our guides and tutorials about IP tracking. Whether you're just figuring this out or already know the basics, there should be something useful here.
