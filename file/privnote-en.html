ğŸ§ Technical Principle (Skip if not tech-savvy)

Zero-width characters include: zero-width space (U+200B), zero-width non-breaking space (U+FEFF), zero-width joiner (U+200D), etc. Privnote.chat converts your secret message to binary data, then uses these invisible characters to represent 0s and 1s, finally inserting them into normal text. Decryption reverses this process.

Simply put: visible text is for others to see, invisible text is for those who know. Like invisible ink, it's magical!
